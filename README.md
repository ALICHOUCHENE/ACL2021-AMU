# Lara Croft X Pacman

`Lara Croft X Pacman` is a labyrinth game developed with Java.


![demo](demo.png)


## Installation
### Prerequisites
You need [Maven](https://maven.apache.org/) to run this game.

## How to play

```bash
mvn compile
mvn exec:java -Dexec.mainClass=start.Main
```
